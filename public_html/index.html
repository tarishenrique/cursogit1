Aprendendo a utilizar o Git e o Github

Comando Função
git init Adicionar pasta ao github
git status Verificar status dos arquivos no git
git add . Adicionar todos os arquivos da pasta ao git
git commit -m "commit inicial" Criar commit (nova versão) no git
git config --global user.email "email@email.com" Adiciona email de acesso ao Github
git config --global user.name "Taris Henrique" Adiciona nome que ficará registrado no commit
git push Subir a versão do repositório para o Github
git remote add origin https://github.com/tarishenrique/xxxxxxx Adcionar para qual repositório o upload será realizado
git reflog Visualizar versão que já foram lançadas no git
git reset --hard idxxxx Alternar a pasta atual para as versões já existentes no git
git branch Visualizar branchs existentes e qual a branch que está sendo utilizada no momento
git branch checkout xxxxx Alternar entre branchs existentes
